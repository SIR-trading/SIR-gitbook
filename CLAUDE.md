# CLAUDE.md

## Project Overview

This is the **SIR Protocol** GitBook documentation site. SIR is a DeFi protocol for safer leveraged trading that eliminates volatility decay and liquidation risks. SIR is deployed on Ethereum, HyperEVM, and MegaETH.

## Structure

- `SUMMARY.md` — GitBook table of contents and navigation structure
- `README.md` — Landing page / introduction
- `protocol-overview/` — All documentation (tokenomics, oracle, security, deployments, leverage mechanics)

## Conventions

- All content is Markdown (`.md`) formatted for GitBook
- `SUMMARY.md` defines page hierarchy and ordering — update it when adding/removing pages
- Pages use GitBook-flavored Markdown (supports `<figure>`, hints, tabs, etc.)
- LaTeX math uses `$$` delimiters
- Emojis are used in page titles within `SUMMARY.md`

## Multi-Chain Deployments

- **Ethereum** — Uniswap V3, SIR token, WETH dividends
- **HyperEVM** — HyperSwap, HyperSIR token, WHYPE dividends
- **MegaETH** — Kumbaya, MegaSIR token, WETH dividends
